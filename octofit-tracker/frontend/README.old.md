# Octofit Tracker Frontend

All frontend React app commands (npm start, npm install, etc.) must be run inside the `octofit-tracker/frontend/octofit-app` directory, since that is where the `package.json` is located.

## Usage

```
cd octofit-tracker/frontend/octofit-app
npm start
```

For installing packages:
```
cd octofit-tracker/frontend/octofit-app
npm install <package-name>
```

Do not run npm or yarn commands in the parent `frontend` directory.
